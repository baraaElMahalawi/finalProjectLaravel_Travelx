توثيق مشروع نظام إدارة حجوزات الفنادق (Travelx)
1. مقدمة عن المشروع
Travelx هو منصة ويب متكاملة تهدف إلى تسهيل عملية إدارة وحجز الغرف الفندقية. يعمل النظام كحلقة وصل بين إدارة الفندق (Admin) والعملاء، حيث يوفر واجهة تحكم شاملة للإدارة لعرض الخدمات المتاحة، وواجهة مستخدم سلسة للعملاء لاستكشاف الخيارات المتاحة وإتمام الحجز إلكترونياً.
2. الميزات والوظائف (Features)
أولاً: لوحة تحكم المسؤول (Admin Dashboard)
 تسجيل الدخول: واجهة خاصة وآمنة للمسؤول.
 إدارة المحتوى: إضافة وتعديل وحذف الغرف والفنادق المتاحة.
 نظام التنبيهات: استقبال رسائل فورية عند إتمام أي عملية حجز جديدة.
 تحديث التوافر: يتم تحديث حالة الغرف في النظام تلقائياً لمنع حدوث تعارض في الحجوزات (Double Booking).
ثانياً: واجهة المستخدم (User Interface)
 نظام العضوية: إمكانية إنشاء حساب باستخدام البريد الإلكتروني (Gmail) وكلمة المرور، مع حفظ البيانات وتشفيرها في قاعدة البيانات.
 التصفح والاستكشاف: استعراض كافة الفنادق والغرف المتاحة مع عرض تفاصيلها الكاملة.
 نظام الحجز المرن: اختيار الغرفة وتحديد تاريخ الحجز المناسب.
 تعدد خيارات الدفع: توفير خيار الدفع الإلكتروني (Visa) أو الدفع النقدي عند الوصول.
3. التقنيات المستخدمة (Technical Stack)
تم بناء المشروع باستخدام أحدث التقنيات لضمان الأداء والأمان:
 الإطار البرمجي (Framework): Laravel (PHP) لبناء المنطق البرمجي وإدارة قاعدة البيانات.
 واجهة التطبيقات البرمجية (API): لربط المكونات وضمان سلاسة انتقال البيانات.
 التصميم وتجربة المستخدم (UI/UX): تم التركيز على واجهات بديهية وجذابة لتوفير أفضل تجربة للمستخدم.
4. دورة عمل النظام (Workflow)
1 الإدارة: يقوم الـ Admin بإدخال بيانات الغرف وصورها وتفاصيلها عبر لوحته الخاصة.
2 المستخدم: يقوم العميل بإنشاء حساب أو تسجيل الدخول، ثم يتصفح الغرف.
3 عملية الحجز: يختار العميل الغرفة، يحدد التاريخ، ويختار وسيلة الدفع المناسبة.
4 التأكيد: بمجرد التأكيد، يتم إرسال إشعار للإدارة، ويقوم النظام فوراً بحجز "الرصيد" (الغرفة) في قاعدة البيانات لمنع حجزها من قبل عميل آخر في نفس الوقت.
