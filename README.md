
 مشروع نظام إدارة حجوزات الفنادق (Travelx)

 وصف المشروع
منصة ويب متكاملة تهدف إلى تسهيل عملية إدارة وحجز الغرف الفندقية. يعمل النظام كحلقة وصل بين **إدارة الفندق (Admin)** والعملاء، حيث يوفر واجهة تحكم شاملة للإدارة لعرض الخدمات المتاحة، وواجهة مستخدم سلسة للعملاء لاستكشاف الخيارات المتاحة وإتمام الحجز إلكترونياً.


  التقنيات المستخدمة
تم بناء هذا المشروع باستخدام أحدث التقنيات لضمان الأداء العالي والأمان البرمجي:
*Framework:* Laravel (PHP) لبناء المنطق البرمجي وإدارة قاعدة البيانات الخلفية.
*Database:* MySQL لحفظ البيانات وتداولها.
*UI/UX Design:* الاعتماد على واجهات بديهية وجذابة لتوفير أفضل تجربة مستخدم وتسهيل التصفح.

خطوات التشغيل :

لوحة تحكم المسؤول
يتمتع المسؤول بكامل الصلاحيات لإدارة العمليات التشغيلية للفندق:
إدارة الغرف والفندق: صلاحية كاملة لإضافة غرف جديدة، تعديل بيانات الغرف الحالية، أو حذفها نهائياً من النظام.
إدارة الحجوزات: استقبال طلبات الحجز المقدمة من العملاء، مع إمكانية القبول (Confirm) أو الرفض/الإلغاء (Cancel) بناءً على حالة توفر الغرف.
إدارة المستخدمين: استعراض ومتابعة قائمة جميع الحسابات المسجلة على المنصة.


 2. بوابة المستخدم/العميل
تتيح للزوار والعملاء تجربة حجز سلسة ومتكاملة:
إدارة الحساب: إمكانية تسجيل حساب جديد (Register) أو تسجيل الدخول (Login) مع تفعيل ميزة "حفظ تسجيل الدخول" (Remember Me).
تصفح واستكشاف الغرف: استعراض تفصيلي لجميع الغرف المتاحة في الفندق، ومعرفة الأسعار والخدمات والميزات الملحقة بكل غرفة.
إدارة الحجوزات الشخصية: إرسال طلبات حجز الغرف مباشرة، ومتابعة حالة الحجوزات الخاصة به عبر لوحة التحكم الشخصية (Dashboard).


ملاحظة: تم إنشاء هذا الموقع في بداية سنة 2025، وتمت إضافة بعض التعديلات والتحديثات البرمجية الحالية ليتناسب تماماً مع متطلبات الواجب الدراسي لعام 2026.
